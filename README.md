# Backtesting with strategies using multiple indicators

1. 趋势类（Trend）-> 用来判断方向
- Simple Moving Average (SMA)
- Exponential Moving Average (EMA)

2. 动量类（Momentum）-> 判断“涨太多/跌太多”

非常重要（比均线更“聪明”）：
- Relative Strength Index (RSI)
- Moving Average Convergence Divergence (MACD)

3. 波动率（Volatility）-> 判断市场是否剧烈波动

控制风险：
- Bollinger Bands
- ATR（Average True Range）

4. 成交量（Volume）-> 判断“上涨是否有量支持”
- Volume MA
- VWAP
